## Technical Specifications
- Image Size: 512x512 pixels
- Format: PNG with transparent background
- Style: Apple Memoji style
- Resolution: High quality (2x for retina displays)
- Color: Full color with good contrast
- File naming: Exactly as specified in the mapping

## Design Guidelines
1. Keep expressions professional but approachable
2. Use role-appropriate accessories
3. Maintain consistent style across all Memojis
4. Ensure good visibility at smaller sizes
5. Use clear, recognizable features for each role
6. Include appropriate industry-standard tools/props
7. Consider diversity in representation
8. Maintain high contrast for accessibility
