# Professional Avatar Specifications

## Design Guidelines for All Avatars
- Use high-quality, professional 3D character renders
- Maintain consistent art style across all avatars
- Focus on facial expressions and upper body
- Use clean, professional backgrounds
- Size: 512x512px minimum
- Format: PNG with transparency
- Include diverse representation

## Technical Requirements
1. Image Resolution: 1024x1024px (can be scaled down)
2. Format: PNG with transparency
3. Color Profile: sRGB
4. File Size: Optimize for web (max 500KB)
5. DPI: 72 for web use
6. Style: 3D rendered or high-quality vector
7. Background: Optional with transparency
